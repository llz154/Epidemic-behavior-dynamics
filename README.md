Code for the paper "Emergent dilemma and periodic oscillation in the nonlinear interplay between epidemic and behavior"

The project utilizes both Python (Jupyter Notebooks) and Mathematica to perform numerical calculations and plot the results.

Contents & Usage
Figure reproduction: run the ".ipynb" files using Jupyter Notebook; run the '.nb' files using Wolfram Mathematica.
Simulation data: run the '.py' files using Python for generating simulation data in well-mixed and networked population.
